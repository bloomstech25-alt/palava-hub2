4. VIEWPORT FIXES
=========================

Ensure the application includes the proper viewport configuration:

<meta name="viewport" content="width=device-width, initial-scale=1, viewport-fit=cover">

Also ensure:
- body {
    margin: 0;
    overflow-x: hidden;
}
- html {
    overflow-x: hidden;
    width: 100%;
}
- Use box-sizing: border-box globally.
