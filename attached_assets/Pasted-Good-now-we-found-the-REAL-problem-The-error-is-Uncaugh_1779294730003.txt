That means your app is trying to use:

something.map(...)

…but that something is NOT an array.

Since it is happening on ALL tabs, your app is probably receiving:

an object instead of array
null
undefined
Firebase snapshot object
Firestore document object

instead of:

[]

You probably have code like:

{}

[]

VERY COMMON FIREBASE MISTAKE

You may be doing this:

const data = doc.data();
setSchools(data);

But data is an OBJECT.

Then later:

which crashes.

CORRECT FIRESTORE ARRAY FETCH

If fetching many documents:

const querySnapshot = await getDocs(collection(db, "schools"));

const schoolsArray = querySnapshot.docs.map(doc => ({
  id: doc.id,
  ...doc.data()
}));

setSchools(schoolsArray);
